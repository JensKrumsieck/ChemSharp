# Chem# (ChemSharp)

[![Maintainability](https://api.codeclimate.com/v1/badges/bb81db40213cc68deb97/maintainability)](https://codeclimate.com/github/JensKrumsieck/ChemSharp/maintainability)

### Supported Filetypes
* Molecule
	* XYZ
	* CIF (crystallographic information file)
	* MOL2 (TRIPOS Mol2)

* Spectroscopy
	* Varian/Agilient DSW
	* Bruker EMX SPC/PAR
